# EZ Memory – Học Ít Hơn, Nhớ Lâu Hơn, Hiệu Quả Hơn

🌐 **Live site:** https://ezmemory.netlify.app/

Dự án website giới thiệu và đăng ký khóa học kỹ năng học tập **EZ Memory** – thuộc môn **SSA101**, Trường Đại học FPT, Nhóm 6.

---

## 📂 Cấu trúc thư mục

```
EzMemory/
├── index.html        # Trang chủ
├── khoa-hoc.html     # Nội dung khóa học
├── dang-ky.html      # Form đăng ký
├── cam-on.html       # Trang cảm ơn sau đăng ký
├── css/
│   └── style.css     # Toàn bộ stylesheet
├── js/
│   └── main.js       # JavaScript (navbar, form validation, animations...)
├── images/           # Ảnh đại diện giảng viên
├── favicon.ico
├── robots.txt
└── sitemap.xml
```

---

## ✨ Tính năng

- **Form đăng ký** với validation client-side, gửi dữ liệu lên Google Sheets qua Apps Script
- **Responsive** – hỗ trợ mobile/tablet/desktop
- **Animations** – fade-in khi scroll bằng IntersectionObserver
- **FAQ & Accordion** – mở/đóng nội dung buổi học
- **Counter animation** – đếm số liệu thống kê
- **Cookie banner** – lưu trạng thái vào localStorage
- **SEO** – đầy đủ meta tags, Open Graph, Twitter Card, JSON-LD Schema

---

## 🚀 Cách chạy local

Dự án là static HTML, chỉ cần mở file trong trình duyệt hoặc dùng live server:

```bash
# Dùng VS Code Live Server extension, hoặc:
npx serve .
```
